# privacy-policy
[바로메모 개인정보처리방침]

1. 개인정보처리방침이란? '바로메모'(이하 ‘앱’)는 이용자의 개인정보를 소중히 다루며,
   관련 법령을 준수합니다. 본 방침은 앱 이용 시 어떤 정보가 어떻게 다루어지는지 이용자에게 투명하게 알리는 지침입니다.

2. 서버 저장 없는 로컬 서비스 (개인정보 수집 안 함) 바로메모는 이용자가 작성한 메모 내용을 외부 서버로 전송하거나 저장하지 않습니다.

   메모 데이터: 이용자가 작성한 모든 메모와 설정값은 이용자의 **스마트폰 내부 저장소(shared_preferences)**에만 저장됩니다.
   데이터 파기: 앱을 삭제하면 기기 내 저장된 모든 데이터는 즉시 삭제되며, 개발자를 포함한 그 누구도 복구할 수 없습니다.

3. 광고 제공을 위한 식별 정보 이용 앱은 무료 서비스 유지 및 광고 게재를 위해 최소한의 식별 정보를 활용합니다.

    항목: 광고 식별자(ADID/IDFA), 기기 정보(모델명, OS 버전)
    목적: Google AdMob(구글 애드몹)을 통한 앱 내 광고 게재
    참고: 광고 식별 정보는 구글(Google)의 광고 정책에 따라 수집 및 관리되며, 이용자는 기기 설정에서 광고 추적을 차단할 수 있습니다.

4. 기기 권한의 사용 앱의 기능을 위해 다음과 같은 기기 권한을 필요로 합니다.

    사진/미디어 접근: 메모 내 사진 첨부 기능을 위해 사용됩니다. (사진 파일을 서버로 전송하지 않으며 기기 내 파일 경로만 참조합니다.)
    알림: 이용자가 설정한 시간에 맞춰 알림을 보내기 위해 사용됩니다.

5. 이용자의 권리 이용자는 언제든지 앱 삭제를 통해 저장된 모든 정보를 스스로 파기할 수 있으며, 기기 설정을 통해 부여된 권한을 철회할 수 있습니다.

6. 연락처 방침과 관련하여 궁금하신 점은 개발 담당자에게 연락 주시기 바랍니다.
